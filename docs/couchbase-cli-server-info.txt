couchbase-cli-server-info(1)
============================

NAME
----
couchbase-cli-server-info - Displays information and statistics about
the Couchbase Server specified.

SYNOPSIS
--------
[verse]
'couchbase-cli server-info' [--cluster <url>] [--username <username>]
	  [--password <password>]

DESCRIPTION
-----------

This command prints information about the Couchbase Server specified.

-------

include::part-common-options.txt[]

include::part-host-formats.txt[]

EXAMPLES
--------

To get information about the Couchbase Server on localhost (127.0.0.1) run
the following command:

	$ couchbase-cli server-info -c 127.0.0.1 --username Administrator \
	 --password password
